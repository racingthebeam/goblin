# Public Block Types

Public block type ID ranges are requestable in batches of 256; to request a range, submit a PR!

  - `0x8000_0000` - `0x8000_03FF`: Goblin built-in block types
  - `0x8000_0400` - `0x8000_07FF`: BEAM256 & associated tooling ([@racingthebeam/beam256](https://github.com/racingthebeam/beam256))
